## About

Senior Software Engineer based in the Philippines, focused on full-stack web and mobile delivery — from data model to deployed UI. Core strength is shipping production-grade products on **React, Laravel, and TypeScript**, with an emphasis on clean architecture, sane state management, and code that's still easy to read six months later.
